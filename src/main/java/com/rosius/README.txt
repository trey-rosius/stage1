
=====-----Popular Movies Api Key---============

In order to run this project, please navigate to this(PopularMovies\app\src\main\java\com\rosius\popularmovies\config\Config.java) directory and add the below line

 public static final String API_KEY = "YOUR-API_KEY";
 
 Please replace YOUR-API_KEY with the API_KEY you get from themoviedb.org
